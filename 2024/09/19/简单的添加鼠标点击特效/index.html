简单的添加鼠标点击特效

2024/9/19 程序猿 javascript

第一步：下载.js文件

我是从这个博主星梦启航的github中获取的js文件，他这里有很多非常有趣的项目以及教程！
两个文件名，一个是 firework.js，一个是 anime.min.js。
确保你的 anime.min.js 和 fireworks.js 文件已经放在了 themes/particlex/source/js/ 目录下

第二步：找到layout.ejs文件

这里需要注意的是我现在所找到的教程或者通过问GPT方式提供的都是footer.ejs文件或者head.ejs文件，这里需要找一下有<head>标签的.ejs文件，所以我找到的是layout.ejs文件。不过我觉得这应该是根据你所用的主题有关系。我用的是particlex这个主题，所以最终找到的文件位置是：\themes\particlex\layout\layout.ejs

第三步：在<head>部分引入js文件

在layout.ejs文件的<head>部分，添加你的js文件引入代码。修改如下： 

<head>
    <meta charset="utf-8" />
    <title><%= title %></title>
    <meta name="author" content="<%- config.author %>" />
    <meta name="description" content="<%- config.description %>" />
    <meta name="keywords" content="<%- config.keywords %>" />
    <meta
        name="viewport"
        content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=0"
    />
    <link rel="icon" href="<%- url_for(theme.avatar) %>" />
    <%- partial("import", { type }) %>
    <!-- 添加这两行代码引入 JS 文件 -->
    <script src="<%- url_for('/js/anime.min.js') %>"></script>
    <script src="<%- url_for('/js/fireworks.js') %>"></script>
</head>
